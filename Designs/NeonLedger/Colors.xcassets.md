# Neon Ledger Color Assets

## Ready-to-Paste Contents.json Entries

Paste these into your `.xcassets` color set folders.

### Background

```json
{
  "colors" : [
    {
      "color" : {
        "color-space" : "srgb",
        "components" : {
          "alpha" : "1.000",
          "blue" : "0x0A",
          "green" : "0x0A",
          "red" : "0x0A"
        }
      },
      "idiom" : "universal"

          "value" : "dark"
```
